### The Philosophy behind EasyPub

EasyPub is an app made by Counterfoil Lab. Counterfoil is dedicated to making "Convivial"  tools: tools that can be easily understood by yeoman developers and that contribute to more empowered citizens. 

Books are a core part of a convivial ecosystem: in so many ways, they serve as the documentation and operating system for free societies. There can be no empowered citizenry without free and open production, sharing, and storing of books. 

We believe that EPUB—an open-source file type that is supported by a plethora of e-reading applications—is a valuable community resource. This file format is a "convivial" tool in two aspects: (1) as a way to share information freely, and (2) as an understandable tool that communities can use insupportable of sharing. As such, we seek to contribute to the ecosystem of people who create and distribute EPUBs. 

Specifically, EasyPub is an easy-to-use ePUB editor. We know that a lot of ePUB conversions through tools like Calibre result in poorly formatted ePUBs. We hope that EasyPub will enable users to edit these outputs to create well-formatted ePUBs. 

We encourage users who have reformatted ePUBs to share them. Of course, we only encourage the sharing of public domain ebooks :)
